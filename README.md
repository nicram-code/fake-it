# Fake Data Generator

Generator fałszywych danych osobowych dla Wielkiej Brytanii i Polski.

## 🌐 Moja strona

Sprawdź moją stronę: [nicram-code.github.io](https://nicram-code.github.io/)

## 📋 Wymagania

- Python 3.7 lub wyższy

## 🚀 Instalacja

1. Sklonuj repozytorium:
```bash
git clone <repozytorium>
cd fake-data-gen
```

2. Utwórz wirtualne środowisko (opcjonalne, ale zalecane):
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# lub
venv\Scripts\activate  # Windows
```

3. Zainstaluj wymagane pakiety:
```bash
pip install -r requirements.txt
```

## 🎮 Użycie

Uruchom program:
```bash
python main.py
```

Po uruchomieniu zobaczysz menu z opcjami:
- **[1] UK** - generuje fałszywe dane dla Wielkiej Brytanii
- **[2] PL** - generuje fałszywe dane dla Polski
- **[S] Site** - otwiera stronę autora w przeglądarce
- **exit** - zamyka program
